# 💡 Ideia e Contexto - Servidor de Manutencao Auditavel

**📂 Categoria:** Operacao / Seguranca

### ❌ O Problema
MVP de servidor de manutencao corporativa com trilha de auditoria, catalogo e HTTPS/mTLS.

### 🚀 A Solução
Registra e controla solicitacoes de manutencao com evidencias operacionais.

### 📈 Valor para o Portfólio
Controla quem pediu acesso, motivo, maquina, chamado, acao, pacote, decisao e job gerado.
