# 💡 Ideia e Contexto - Cartas Vivas Streamlit

**📂 Categoria:** Aplicacao web

### ❌ O Problema
Versao Python/Streamlit de um devocional RPG Cristao.

### 🚀 A Solução
Exibe devocionais por capitulo com filtros e fontes de contingencia.

### 📈 Valor para o Portfólio
Oferece leitura diaria deterministica, sorteio de capitulos, filtros por testamento, versoes biblicas locais e contingencia de API.
