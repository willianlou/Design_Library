# 💡 Ideia e Contexto - Sistema de Agendamento

**📂 Categoria:** Backend

### ❌ O Problema
MVP de agendamento com controle de concorrencia e prevencao de dupla reserva.

### 🚀 A Solução
Cria reservas e retorna conflito quando o horario ja esta ocupado.

### 📈 Valor para o Portfólio
O projeto usa transacoes e constraints de banco para impedir que duas pessoas reservem o mesmo profissional no mesmo horario.
