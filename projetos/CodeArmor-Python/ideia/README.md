# 💡 Ideia e Contexto - Protetor de Apps Python

**📂 Categoria:** Seguranca de software

### ❌ O Problema
Ferramenta local para proteger aplicativos Python com assinatura, licenca e ofuscacao opcional.

### 🚀 A Solução
Empacota controles locais para dificultar alteracao e uso nao autorizado de apps Python.

### 📈 Valor para o Portfólio
Gera certificado ECDSA, assina com SHA-256, aplica guarda de execucao e integra PyArmor quando instalado.
