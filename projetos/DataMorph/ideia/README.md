# 💡 Ideia e Contexto - Virtualizador de Dados GRC

**📂 Categoria:** GRC / Dados

### ❌ O Problema
Aplicacao Streamlit no-code para carregar, transformar, validar e exportar dados.

### 🚀 A Solução
Permite preparar dados GRC sem exigir conhecimento de Pandas ou programacao.

### 📈 Valor para o Portfólio
Aceita CSV, JSON, texto JSON e API REST GET, achata estruturas, permite edicao visual e oferece operacoes sem codigo.
