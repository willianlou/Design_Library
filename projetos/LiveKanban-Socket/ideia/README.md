# 💡 Ideia e Contexto - App de Gestao de Projetos (Kanban)

**📂 Categoria:** Gestao de projetos

### ❌ O Problema
MVP de quadro Kanban com drag-and-drop persistente e colaboracao em tempo real.

### 🚀 A Solução
Permite criar e mover cards em quadros colaborativos.

### 📈 Valor para o Portfólio
O backend e a fonte da verdade para a ordem dos cards e transmite alteracoes para usuarios conectados na mesma sala.
