# 💡 Ideia e Contexto - Buscador Publico Streamlit

**📂 Categoria:** Dados / Privacidade

### ❌ O Problema
Aplicativo local para consolidar mencoes publicas em fontes abertas.

### 🚀 A Solução
Pesquisa fontes abertas e gera relatorio sob demanda no navegador.

### 📈 Valor para o Portfólio
Consulta resultados de busca, concursos, diarios oficiais, processos judiciais e outras fontes, mantendo os dados da busca apenas na sessao.
