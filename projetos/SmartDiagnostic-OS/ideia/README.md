# 💡 Ideia e Contexto - Diagnostico Inteligente TI

**📂 Categoria:** Suporte / Windows

### ❌ O Problema
Super app Windows para triagem e diagnostico tecnico de sistemas, rede, impressora e seguranca.

### 🚀 A Solução
Orienta a investigacao e coleta evidencias tecnicas de problemas de TI.

### 📈 Valor para o Portfólio
Direciona usuarios por perguntas simples, executa verificacoes reais e gera evidencias tecnicas para equipes de TI.
