# 💡 Ideia e Contexto - NetworkSentinel

**📂 Categoria:** Seguranca / Redes

### ❌ O Problema
Ferramenta Windows para monitorar conexoes de rede e destacar trafego suspeito por processo.

### 🚀 A Solução
Exibe conexoes ativas e apoia a investigacao de trafego suspeito.

### 📈 Valor para o Portfólio
Lista conexoes TCP/UDP, associa processos, categoriza portas, importa logs JSON/JSONL e grava sessoes de escuta.
