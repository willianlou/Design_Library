# 💡 Ideia e Contexto - CyberRank Phishing Academy

**📂 Categoria:** Conscientizacao / Seguranca

### ❌ O Problema
Aplicacao Streamlit para ensinar usuarios a identificar tentativas de phishing.

### 🚀 A Solução
Treina e avalia a capacidade de reconhecer phishing.

### 📈 Valor para o Portfólio
Oferece trilhas, exemplos guiados, quizzes, progressao por nivel e gamificacao para treinamento de usuarios.

## Recursos

- Dashboard de progresso com XP, nivel e rank.
- Sistema de moedas para desbloquear itens digitais simbolicos.
- Loja com temas, avatar, banner e medalhas.
- Tela inicial personalizavel com animacoes em CSS.
- Modulos de treinamento com sinais de alerta.
- Laboratorio com exemplos de mensagens suspeitas.
- Quiz com feedback imediato.
- Simulador para analisar uma mensagem colada pelo usuario.
