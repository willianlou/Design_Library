# 💡 Ideia e Contexto - CyberRank Phishing Academy

**📂 Categoria:** Conscientizacao / Seguranca

### ❌ O Problema
Aplicacao Streamlit para ensinar usuarios a identificar tentativas de phishing.

### 🚀 A Solução
Treina e avalia a capacidade de reconhecer phishing.

### 📈 Valor para o Portfólio
Oferece trilhas, exemplos guiados, quizzes, progressao por nivel e gamificacao para treinamento de usuarios.
